# Task 2 - Phishing Email Analysis

## Overview

This project was completed as part of a Cyber Security Internship task focused on identifying and analyzing phishing emails. The objective was to examine suspicious email samples and identify common phishing indicators used by attackers to deceive users.

## Objective

To analyze phishing email samples and detect characteristics that indicate fraudulent or malicious intent.

## Tools Used

* Text Editor
* Web Browser
* Online Research
* Email Header Analysis Concepts

## Phishing Indicators Identified

The following phishing traits were found during the analysis:

* Suspicious sender email addresses
* Domain spoofing and impersonation
* Fraudulent URLs
* Urgent and threatening language
* Generic greetings
* Social engineering tactics
* Fear-based manipulation
* Requests for sensitive information

## Analysis Process

1. Examined the sender's email address.
2. Reviewed the email content for urgency and threats.
3. Analyzed links for domain mismatches.
4. Checked for impersonation of trusted organizations.
5. Identified social engineering techniques used to manipulate users.
6. Documented all findings in detailed reports.

## Key Concepts Learned

* Phishing
* Email Spoofing
* Social Engineering
* URL Analysis
* Threat Detection
* Email Security Awareness

## Conclusion

Both email samples exhibited multiple phishing indicators and were classified as phishing attempts. This task improved understanding of how attackers use deception techniques to steal sensitive information and compromise user accounts.

## Outcome

Successfully analyzed phishing email samples, identified phishing characteristics, and gained practical knowledge of email threat detection and cybersecurity awareness.
